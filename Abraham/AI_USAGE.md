# Here is every time I used AI:

- I used AI to write a script that downloaded the MRI data from the amazon web server.

- I also used AI to write code to preprocess this data and extract features for further analysis.

- AI helped me layout how to structure my code (what modules I need), and also a draft some comment-psuedocode for some of the algorithms.

- AI rewrote part of my code to implement memory saving methods like checkpointing and chunking in the forward pass of the HyperFuseNet class.

- AI helped me translate some complex math into code. For example, the interbrain patchwise attention torch.einsum line.

- AI helped me debug and make sure all of the data was in the correct shape at every step.
